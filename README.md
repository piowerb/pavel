# Nobody Meditates Here

This is the official repository for the Nobody Meditates Here website, a meditation project founded by Pavel Bielski. 
The technical architecture, code development, and repository maintenance are owned and managed by @piowerb on GitHub.

The project provides simple sitting meditation instructions for students and beginners.

## Features
*   Multilingual Support: The site includes a custom translation system for English, Polish, and Russian.
*   Responsive Design: The interface is optimized for all screen sizes, including mobile and desktop devices.
*   Interactive Components: The site features a video carousel with touch support and a lightbox gallery for images.
*   Optimized Performance: It utilizes Intersection Observer for scroll animations and efficient asset loading.

## Technical Stack
*   Structure: Semantic HTML5.
*   Styling: Modern CSS3 using Flexbox, Grid, and CSS Variables.
*   Interactivity: Vanilla JavaScript for multilingual support and UI logic.
*   Forms: Contact submissions are handled via Formspree integration.
